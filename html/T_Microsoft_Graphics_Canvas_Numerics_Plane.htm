Remarks

When targeting the Universal Windows Platform, this struct is moved to the namespace Windows.Foundation.Numerics.

This Windows Runtime type is only used for data exchange, and does not include any methods for manipulating the values or performing math computations. For more complete functionality, see the .NET System.Numerics and C++ Windows.Foundation.Numerics namespaces.

System.Numerics.Plane and Windows.Foundation.Numerics.plane provide implicit conversion operators to and from Microsoft.Graphics.Canvas.Numerics.Plane, so you can use these richer types in your application code, then pass them directly to Windows Runtime APIs that consume the Microsoft.Graphics.Canvas.Numerics types.
