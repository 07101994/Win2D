Remarks

This Windows Runtime type is only used for data exchange, and does not include any methods for manipulating the values or performing math computations. For more complete functionality, see the .NET System.Numerics and C++ Windows.Foundation.Numerics namespaces.

System.Numerics.Matrix4x4 and Windows.Foundation.Numerics.float4x4 provide implicit conversion operators to and from Microsoft.Graphics.Canvas.Numerics.Matrix4x4, so you can use these richer types in your application code, then pass them directly to Windows Runtime APIs that consume the Microsoft.Graphics.Canvas.Numerics types.
