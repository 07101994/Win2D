Microsoft.Graphics.Canvas.Effects Namespace

This namespace provides image manipulation effects such as blur and saturation.

Effects are a type of ICanvasImage, so they can be displayed using DrawImage Overload or CanvasImageBrush. The source data fed into an effect is also of type ICanvasImage, which allows you to connect either a bitmap or another effect. Complex image processing graphs can be constructed by chaining many effects together.

Applies an arbitrary 2D filter kernel to an image. This can perform operations such as edge detect, emboss, sharpen, or blur.

Supported by Win2D but not Windows.UI.Composition.

CropEffect

Supported by Win2D but not Windows.UI.Composition.

DirectionalBlurEffect

Blurs an image in a particular direction. You can use this effect to make an image look as if it is in motion or to emphasize an animated image.

Supported by Win2D but not Windows.UI.Composition.

Supported by Win2D but not Windows.UI.Composition.

DisplacementMapEffect

Displaces the pixels of an input image by the intensity values of a second image.

Supported by Win2D but not Windows.UI.Composition.

DistantDiffuseEffect

Lighting effect creates an image that appears to be an illuminated non-reflective surface. This effect uses the alpha channel as a height map, and lights it with an infinitely distant light source (such as the sun).

Supported by Win2D but not Windows.UI.Composition.

DistantSpecularEffect

Lighting effect creates an image that appears to be a reflective surface. This effect uses the alpha channel as a height map, and lights it with an infinitely distant light source (such as the sun) applying the specular portion of the Phong lighting model.

Supported by Win2D but not Windows.UI.Composition.

DpiCompensationEffect

Scales an input bitmap to match a different DPI. This is useful for situations where a bitmap is created or loaded at a different DPI than the screen.

Supported by Win2D but not Windows.UI.Composition.

GammaTransferEffect

Alters the colors of an image by applying a per-channel gamma transfer function.

Supported by Win2D but not Windows.UI.Composition.

GaussianBlurEffect

Creates a blur effect based on the Gaussian function over the entire input image.

Supported by Win2D but not Windows.UI.Composition.

HueRotationEffect

Alters the color of an image by rotating its hue values.

Supported by Win2D but not Windows.UI.Composition.

LinearTransferEffect

Alters the colors of an image by applying a per-channel linear transfer function.

Supported by Win2D but not Windows.UI.Composition.

LuminanceToAlphaEffect

Generates an image with alpha channel set to the luminance of the source image, and color channels set to 0. Use it to make a semitransparent overlay based on the brightness of the source image.

Supported by Win2D but not Windows.UI.Composition.

MorphologyEffect

Thins or thickens edge boundaries in an image. This effect uses a filter kernel that is 2 times the size you specify. The kernel is centered on the pixel being calculated, and returns the maximum value in the kernel (if dilating) or minimum value (if eroding).

Supported by Win2D but not Windows.UI.Composition.

OpacityMetadataEffect

Marks an area of an input image as opaque, which can enable internal rendering optimizations.

Supported by Win2D but not Windows.UI.Composition.

PointDiffuseEffect

Supported by Win2D but not Windows.UI.Composition.

ShadowEffect

Lighting effect creates an image that appears to be a reflective surface where the light source is limited to a directed cone of light. This effect uses the alpha channel as a height map, and lights it with a point light source applying the specular portion of the Phong lighting model.
