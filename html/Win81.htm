Windows 8.1 support
Windows 8.1 support

Win2D can be used to build Windows 8.1 and Windows Phone 8.1 apps.

Substituted Types

Some types that are included as part of the Universal Windows Platform are not available on Windows 8.1. Where possible, Win2D provides its own implementation of these.

Numerics Types

For the Universal Windows Platform, the System.Numerics namespace is implemented by the CLR assembly System.Numerics.Vectors. The CLR assembly is able to provde more efficient interop with WinRT APIs. In addition, the runtime is able to perform SIMD optimizations on code using these types.

Win2D for Windows 8.1 provides an implementation of the System.Numerics namespace in the Microsoft.Graphics.Canvas.Numerics.dll assembly.

As a general rule, if you code against System.Numerics then the same code will build with all flavors of Win2D.
