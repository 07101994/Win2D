  Member nameValueDescription
Default0[Experimental] Automatically positioned, if LineSpacing is negative; otherwise, LineSpacing and LineSpacingBaseline are treated as a quantities in DIPs.
Uniform1[Experimental] Spacing is affected by the absolute value of LineSpacing, and vertical offset of the text is determined by LineSpacingBaseline. LineSpacing and LineSpacingBaseline are treated as a quantities in DIPs.
Proportional2[Experimental] Spacing is affected by the absolute value of LineSpacing, and vertical offset of the text is determined by LineSpacingBaseline. LineSpacing and LineSpacingBaseline are treated as a ratio of the default.
Remarks

This API is marked as [Experimental], meaning it may be subject to change in future Win2D releases.

This API is available on Windows 10 (Universal Windows Platform) only. It is not supported on Windows 8.1 or Phone 8.1.
