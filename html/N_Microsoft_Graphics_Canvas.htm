Microsoft.Graphics.Canvas Namespace
This namespace defines the main Win2D API for drawing 2D graphics.
Classes
  ClassDescription
CanvasAnimatedControl
XAML control providing swap chain rendering. Useful for scenarios with large amounts of quickly animating content.
CanvasAnimatedDrawEventArgs
Provides data for the Draw event.
CanvasAnimatedUpdateEventArgs
Provides data for the Update event.
CanvasBitmap
A bitmap is a 2D grid of pixels that form an image.
CanvasCommandList
A list of commands that can be recorded and used anywhere an image can be used.
CanvasControl
XAML control providing immediate mode 2D rendering. Start here if you are new to the Win2D API.
CanvasCreateResourcesEventArgs
Provides data for the CreateResources event.
CanvasDevice
Devices are used to create graphics resources and drawing sessions. Most apps should use the CanvasControl wrapper instead of directly managing their own CanvasDevice.
CanvasDrawEventArgs
Provides data for the Draw event.
CanvasDrawingSession
Drawing sessions are used to issue graphics drawing commands. This is the main way to draw things onto a canvas.
CanvasImageBrush
A brush which paints using image content, such as a bitmap. Useful for producing tiled backgrounds, or filling geometry with a pattern.
CanvasImageSource
Derives from Windows.UI.Xaml.Media.Imaging.SurfaceImageSource and extends it to know about Canvas types. Most apps should use CanvasControl instead of directly managing their own CanvasImageSource.
CanvasLinearGradientBrush
A brush used to paint a linear gradient.
CanvasRadialGradientBrush
A brush used to paint a radial gradient.
CanvasRenderTarget
A rendertarget is a bitmap that can be drawn onto.
CanvasSolidColorBrush
The simplest kind of brush. Paints things a solid color.
CanvasStrokeStyle
Defines a stroke style for drawing lines. Dashes, dots, how to join segments, how to cap the ends, etc.
CanvasSwapChain
A swap chain is a series of buffers that can be drawn to and presented to the screen.
CanvasSwapChainPanel
A swap chain panel is a XAML control which draws to a swap chain.
CanvasTextFormat
Describes font and layout options for drawing text.
Structures
  StructureDescription
BitmapSize
Describes the size of a bitmap, in integer pixel units.
CanvasGradientStop
Used to describe one gradient stop in a gradient brush.
CanvasTimingInformation
Contains information about a CanvasAnimatedControl's timer.
Interfaces
  InterfaceDescription
ICanvasAnimatedControl
XAML control providing swap chain rendering. Useful for scenarios with large amounts of quickly animating content.
ICanvasBrush
A brush defines the color of things being drawn. It can be a solid color, or a bitmap pattern, or a gradient.
ICanvasImage
Abstract interface representing anything that stores or is able to create a 2D image. This can be a CanvasBitmap, command list, or image processing effect.
ICanvasResourceCreator
Represents any object that can create graphics resources. Implemented by CanvasControl, CanvasDevice, and CanvasDrawingSession.
ICanvasResourceCreatorWithDpi
Represents an object that can create graphics resources, and which has a specific DPI. Implemented by CanvasControl and CanvasDrawingSession.

CanvasComposite
The composite mode used for the effect.
CanvasCreateResourcesReason

CanvasDashStyle
Describes the sequence of dashes, dots, and gaps in a stroke style.
CanvasDebugLevel
Indicates the type of information provided when the Direct2D Debug Layer is enabled.
CanvasDrawTextOptions
Specifies text drawing options.
CanvasEdgeBehavior
Specifies the way in which a brush paints areas outside its typical content region.
CanvasHardwareAcceleration
Specifies whether a graphics device uses GPU hardware acceleration or software rendering.
CanvasImageInterpolation
Specifies the quality used for image resizing and transforms.
CanvasLineJoin
Describes the shape that joins two lines or segments.
CanvasLineSpacingMethod
Specifies how text line spacing is applied.
CanvasStrokeTransformBehavior
Defines how the world transform, dots per inch (DPI), and stroke width affect the shape of the pen used to stroke a primitive.
CanvasSwapChainRotation

CanvasTextAntialiasing
Specifies the antialiasing mode for drawing text.
CanvasTextDirection
Specifies the direction in which text is formatted and read.
CanvasTextTrimmingGranularity
Specifies the granularity used to trim text overflowing the layout box.
CanvasUnits
Specifies what units coordinates are in. This is important for making sure everything stays in the right place when running on high DPI devices.
CanvasVerticalAlignment

CanvasWordWrapping
