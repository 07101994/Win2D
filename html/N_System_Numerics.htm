System.Numerics Namespace

This namespace defines .NET vector and matrix types. It extends the structs from Microsoft.Graphics.Canvas.Numerics with a range of mathematical operators and functions.

This namespace is only available in .NET. Its C++ equivalent is Windows.Foundation.Numerics.
