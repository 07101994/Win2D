Remarks
This matrix type uses a row vector layout. Matrices can be either row vector or column vector. Row vector matrices view vectors as a row from left to right, while column vector matrices view vectors as a column from top to bottom. For example, the x and y of this matrix's translation vector correspond to the fields M31, M32.
